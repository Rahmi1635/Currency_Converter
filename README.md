
# 💱 Canlı Döviz Çevirici (Python & Tkinter)

Bu proje, Python'un **Tkinter** kütüphanesi kullanılarak geliştirilmiş, 160'tan fazla para birimi arasında anlık dönüşüm yapabilen modern bir masaüstü uygulamasıdır. Veriler, **ExchangeRate-API** aracılığıyla gerçek zamanlı olarak çekilmektedir.



## 🚀 Özellikler

- **Anlık Veri:** Dünya çapında 160'tan fazla para birimi için en güncel kurlar.
- **Kullanıcı Dostu Arayüz:** Sade, anlaşılır ve kullanımı kolay tasarım.
- **Çapraz Kur Desteği:** Sadece USD odaklı değil, her para birimini birbiri arasında çevirebilme özelliği.
- **Ters Çevirme (Swap):** Tek butonla kaynak ve hedef para birimlerini yer değiştirme.
- **Hata Yönetimi:** Geçersiz miktar girişlerinde veya bağlantı sorunlarında kullanıcıyı bilgilendirme.

## 🛠️ Kurulum ve Çalıştırma

Projeyi yerel bilgisayarınızda çalıştırmak için aşağıdaki adımları izleyebilirsiniz:

### 1. Kullanılan Kütüphaneler
- `requests` (API verilerini çekmek için)
- `tkinter` (Arayüz tasarımı için)
- `json` (Veri formatını işlemek için)
